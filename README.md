# Automated Abstractive Text Summarization

An AI-powered text summarization system built with Streamlit and Hugging Face Transformers.

## Features

- Abstractive text summarization
- Facebook BART Large CNN model
- TXT, PDF, DOCX upload
- Manual text input
- Summary length selection
- Download summaries
- Word count metrics
- Compression percentage
- Reading time estimation
- Interactive charts

## Tech Stack

- Python
- Streamlit
- Hugging Face Transformers
- PyTorch
- Plotly
